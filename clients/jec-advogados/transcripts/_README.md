# Transcrições de vídeos

Coloque aqui transcrições completas de vídeos já publicados pelo cliente (um arquivo por vídeo).

**Recomendado:** nome descritivo (ex: `whisky-busca-apreensao-ep01.md`).

**Conteúdo:** texto falado no vídeo, do começo ao fim. Pode ser bruto (sem formatação) ou já editado.

Esses transcripts servem pra ensinar o Claude o **tom de fala** real do cliente — pausas, expressões, jeito de explicar, frases recorrentes. Quanto mais transcripts, mais natural o roteiro gerado vai soar.

Pode apagar este `_README.md` depois.
